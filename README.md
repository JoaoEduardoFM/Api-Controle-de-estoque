## Desenvolvimento concluído.

1) Cadastra produto.
2) Edita produto.
3) Exclui produto.
4) Lista produtos cadastrados.
5) Busca registro pelo codigo.
6) Busca registro pela categoria.
7) Efetua baixa de estoque.
8) Efetua entrada de estoque.
